# What is Codon/code-of-conduct?

This repository is used by Project Codon to transparently describe how the review committee enforces the Code of Conduct for users of the Codon. We aim to deliver clarity on our review process by:

* providing insight into how decisions are made and dealt with
* improving our processes with feedback recieved from users of Codon
* providing clarity on how we enforce our code of conduct across Codon repositories

# Contributing to this repository
1. [Fork](https://help.github.com/en/articles/fork-a-repo) codon-code-of-conduct on github.
2. Write your changes/additions in a new branch and update the CHANGELOG.md with the changes you have made.
3. Submit a [pull request](https://help.github.com/en/articles/creating-a-pull-request) to codonPython with a clear description of what you have done.

We suggest that you make sure all of your commits are atomic (one feature per commit).

# Code of Conduct
This repository is under the [Codon Code of Conduct](https://github.com/codonlibrary/codon-code-of-conduct/tree/master). By contributing to it, you agree to comply with it.
